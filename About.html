Features

Basic functionality:

Select a geographical feature from a prepopulated list and recieve a selection of locations from all over the world!

Dinofii will display the most populated region nearest the feature which you search for and will display a WIFI SCORE based on the number of WiFi hotspots found in the nearest postal code. (more on this later)

Types of features include: over 100 options from Streams, Parks, Plateaus, Resorts, and Spas

Using this search the user can research and plan an adventure without the worry of losing connectivity along the way

Stay connected. Stay working. See the world.

Design

Using Html, Bootstrap, and CSS we have designed a minimal and user friendly interface.

We have created a simple search bar with prepopulated features that you can search or browse.

Results are dynamicly displayed on Bootstrap Cards and stack as you continue to search.

Displayed results come with links to map your destination, search for areas with high internet traffic and gather more informtaion.

API Talk

First Api: GEONAMES

Finds global locations (points of interest, not cities) based on a single feature code.

Second Api: EZCMD

Finds postal codes and nearby cities based on latitude and longitude coordinates.

Third Api: WIGLY

Finds the number of WiFi Hotspots in a given countries postal code.

New Technology Used

Wigly API required sending its authentication name:token as a Basic Authorization HTTP header in Base64.
