API Talk

First Api: GEONAMES

Finds global locations (points of interest, not cities) based on a single feature code

Second Api: EZCMD

Finds postal codes and nearby cities based on latitude and longitude coordinates

Third Api: WIGLY

Finds the number of WiFi Hotspots in a given countries postal code

New Technology Used

Wigly API required sending its authentication name:token as a Basic Authorization HTTP header in Base64
